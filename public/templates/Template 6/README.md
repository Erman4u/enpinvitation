# Wedding Invitation - Blacky & Cimeng

A premium, modern, and responsive digital wedding invitation built with **Next.js 15**, **Tailwind CSS**, and **Framer Motion**.

## Tech Stack
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Getting Started

### Local Development
1. Install dependencies:
   ```bash
   npm install --legacy-peer-deps
   ```
2. Run development server:
   ```bash
   npm run dev
   ```
   *Note: On Windows, if you encounter PowerShell core script errors, use `npm.cmd run dev`.*

### Deployment (Vercel)
1. Push this repository to GitHub.
2. Import the project in Vercel.
3. Vercel will automatically detect Next.js and deploy.

## Features
- Fullscreen Opening Screen with blur effects.
- Parallax Hero section.
- Responsive Countdown timer.
- Interactive Love Story timeline.
- Image Gallery with lightbox preview.
- RSVP confirmation via WhatsApp.
- Digital Gift section with copy-to-clipboard functionality.
- Floating Background Music player.
